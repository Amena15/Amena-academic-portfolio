Computer Architecture and Organization-CSN6114

This course covers the fundamental principles of computer system design, focusing on how hardware and software interact. Key topics include instruction sets, memory hierarchy, data paths, and control units, providing students with a solid foundation in computer system functionality.

Lecture Notes

  • lec_a-01 - number systems and codes
  • lec_a-02 - introduction to digital logic and boolean algebra
  • lec_a-03 - design of combinational logic circuits
  • lec_a-04 - standard combinational logic circuits
  • lec_a-05 - sequential logic circuits-flip-flops
  • lec_a-06 - sequential logic circuits-counters and shift registers
  • lec_a-07 - data representation and arithmetic
  • lec_b-01 - Basic Concepts and System Buses
  • lec_b-02 - Structure and Function of Central Processing Unit
  • lec_b-03 - Instruction pipelining and operation of control unit
  • lec_b-04 - Instruction Sets Architecture and Design
  • lec_b-05 - Memory Architecture
  • lec_b-06 - Interfacing input output strategies
  • lec_b-07 - Multiprocessing
